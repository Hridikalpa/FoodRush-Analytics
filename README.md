# FoodRush Analytics

## Overview

FoodRush Analytics is an end-to-end analytics project that simulates the work of a Data Analyst at a food delivery platform similar to Swiggy or Zomato.

The project focuses on answering real business questions using SQL, ETL concepts, Power BI, and Python. Rather than solving isolated SQL exercises, it demonstrates how data is transformed into actionable business insights.

## Business Objectives

* Track revenue, profit, and operational performance
* Analyze customer acquisition channels and marketing effectiveness
* Measure customer retention and repeat purchase behavior
* Identify funnel drop-offs and conversion opportunities
* Understand the impact of discounts on profitability
* Detect high-value customers using Pareto analysis
* Support executive decision-making through dashboards and KPIs

## Tech Stack

* SQL (analysis and KPI computation)
* Power BI (interactive dashboards)
* Python (automation and data preparation)
* ETL concepts (data transformation and reporting pipelines)

## Skills Demonstrated

* Business KPI analysis
* Advanced SQL (CTEs, window functions, joins, subqueries)
* Data modeling and fact/dimension design
* Executive reporting
* Funnel and retention analysis
* Customer segmentation
* Dashboard development
